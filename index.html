RotaSenso Firmware Installation

Web Serial may not be supported in your browser. For best results, use Chrome, Edge, or Opera.

Installation Instructions

  1. Connect your ESP32 to your computer via USB
  2. Click the "Install RotaSenso Firmware" button below
  3. Select the correct COM port when prompted
  4. Wait for the installation to complete
  5. Your RotaSenso will automatically restart with the new firmware

ESP32 Boot Mode Sequence

If you receive an initialization error, follow these steps to put your ESP32 in bootloader mode:

1️⃣
Press and hold the BOOT button (or GPIO0 button)
2️⃣
While holding BOOT, press and release the RESET button
3️⃣
Continue holding BOOT for another second
4️⃣
Release BOOT and click "Install" immediately

Troubleshooting

If you see "Failed to initialize" error:

  • Make sure you're using a data USB cable (some cables are power-only)
  • Try the boot sequence described above
  • Try a different USB port on your computer
  • Install the CP210x USB to UART Bridge VCP Drivers if you haven't already
  • On Windows, check Device Manager to ensure the COM port appears when the device is plugged in
